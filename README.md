# To-Do
A To-Do App for those in school

Name: To-Do App
Description: Tge project is a task manager meant for those in school or those who need help organizing their day to day tasks. 
Installation instructions: You clone the respiratory, install the dependencies, and start the server use while hosting it on localhost:3000
Usage examples: Once the app is running, you can do things such as add a task, complete tasks, edit tasks, deleting tasks, and a whole bunch of other things. 
Project status and roadmap:

  v1.0 - it can create, read, update, and deletre posts. It is running to a user controlled cloud sync.
